# Identification Statement

The policy shock is a high-frequency ECB announcement surprise, aggregated to the month and signed so that positive values represent easing news.

The identifying assumption is timing based. Announcement-window surprises are measured before monthly macro-financial outcomes are realized, so they can be used as external policy-news shocks in the monthly response system.

Information-effect-sensitive events are flagged and checked separately. This matters because ECB announcements may reveal central-bank views about the economy, not only policy decisions.

The thesis estimates reduced-form response paths to policy-news shocks. It does not claim welfare effects, inequality effects, exact bank mediation, exact monthly house-price effects, or exact monthly compensation effects.
