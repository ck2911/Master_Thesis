# Project Map

The repository has four working layers.

## Data Construction

The data scripts build the ECB surprise measures, the monthly research panel, and the lower-frequency background series.

## Estimation

The estimation scripts run first-stage screens, information-effect checks, proxy selection, monthly impulse responses, and mechanism tables.

## Results

The results layer contains the figures and tables used for the thesis: IRFs, cumulative responses, uncertainty checks, robustness checks, and banking-timing evidence.

## Presentation

The notebook and short notes translate the estimates into the thesis argument. Archive material is background only.
