# Final Data Source Registry

This registry covers the active Euro Area / Germany quarterly dataset and the files that feed it. Retired prototypes remain archived for auditability only and are not part of the active pipeline.

| File | Source Institution | Transformation | Frequency | Role in Thesis |

## Active/Archive Firewall

Active code, configuration, and scripts are checked by:

```bash
python3 scripts/check_active_archive_firewall.py
```

The check fails if active operational files reference the retired archive path, retired flat source package, retired notebook pipeline, or legacy data path. Documentation may mention the archive for auditability, but active imports and configuration must remain EU/DE-only.
